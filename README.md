# False Position Method (Regula Falsi)

Java implementation of the False Position (Regula Falsi) numerical method for finding roots of nonlinear equations.

## Description
The False Position method is a bracketing root-finding algorithm that combines the reliability of the Bisection method with faster convergence by using linear interpolation.

This project computes an approximate root of a continuous function within a given interval \([a, b]\) where the function changes sign.

## Features
- Iterative approximation of roots
- Configurable tolerance
- Maximum number of iterations
- Iteration results display
- Error handling when convergence is not achieved

## Input Parameters
- Function \(f(x)\)
- Lower bound \(a\)
- Upper bound \(b\)
- Tolerance
- Maximum iterations
